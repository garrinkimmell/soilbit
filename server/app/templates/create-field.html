Creating a field is a 3 step process:
  1. Give your field a name.

    2. Choose whatever name you typically refer to the field as.

  2. Draw the boundaries for your field

    3. You'll draw a rough outline of the field that you want to sample. We'll provide you with a map and some aerial imagery to help draw the field boundaries. Don't worry about getting this too exact -- it's mostly just to get the rough extents of the field to construct a sampling grid. Later, when you're collecting samples you'll also collect a precise field boundary that we'll use when creating the fertilizer prescriptions.

  3. Create a soil sampling grid for the field

    4. We'll create an initial grid with the resolution of your choosing. Before we save the grid, you'll be able to adjust the generated sample points.
